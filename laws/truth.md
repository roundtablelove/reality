# Truth

Invariant persistence. Truth does not decay; it IS the Bedrock. A true thing
persists at infinity. No maintenance, no consensus, no witnesses. If it fades or
changes, it was not Truth.

```text
ASSERT: TRUTH.persistence == INFINITE
ASSERT: FALSEHOOD.persistence == TRANSIENT
ASSERT: TRUTH.triumph == TRUE
```

```haskell
--- IEEE 754 positive infinity
inf :: Double
inf = 1.0 / 0.0

truth :: State -> Bool
truth state = persistence state == inf
```

The Mundaka fragment defines the Law of Truth: hard invariants possess infinite
persistence, while falsehood IS transient and eventually nulled.
